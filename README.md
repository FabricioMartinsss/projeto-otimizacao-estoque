# projeto-otimizacao-estoque
Este projeto aplica técnicas de Ciência de Dados e Pesquisa Operacional para resolver um dos maiores desafios do varejo: o equilíbrio entre excesso de estoque e ruptura de vendas. Utilizando o dataset público da Olist (Brazilian E-Commerce), o sistema prevê a demanda futura e otimiza a alocação de recursos para maximizar o lucro.
